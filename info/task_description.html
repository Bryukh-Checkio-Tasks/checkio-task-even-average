Our robots teach math and even-odd numbers.

You are given an array of integers. Find all even numbers and calculate the average for they. The result should be rounded down to integer.

Input: A tuple of integers.

Output: The integer.

Example: 

checkio((1, 2, 3, 4, 5, 6)) == 4
checkio((0, 1, 10, 2)) == 4
checkio((-5, -2, 0, 2) == 0
